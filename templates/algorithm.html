SM-2 Spaced Repetition Algorithm

What is SM-2?

SuperMemo 2 (SM-2) is a scientifically-proven algorithm that determines the optimal time to review flashcards. It was developed by Piotr Wozniak in 1988 and is used by Anki and many other spaced repetition systems.

How It Works

The algorithm uses three key parameters for each card:

  • Easiness Factor (EF) - Represents card difficulty (1.3 to 3.0+)
  • Interval (I) - Days until next review
  • Repetitions (n) - Number of successful reviews

Rating System

Again (0)

Completely forgot

→ Reset to 1 day

Hard (1)

Difficult to remember

→ Reset to 1 day

Good (2)

Remembered correctly

→ Increase interval

Easy (3)

Very easy

→ Longer interval (×1.3)

Interval Calculation

if quality < 2:  # Again (0) or Hard (1)
    new_interval = 1 day
    new_repetitions = 0
    
else:  # Good (2) or Easy (3)
    new_repetitions = repetitions + 1
    
    if repetitions == 1:
        new_interval = 1 day
    elif repetitions == 2:
        new_interval = 6 days
    else:
        new_interval = interval × easiness_factor
    
    if quality == 3:  # Easy bonus
        new_interval = new_interval × 1.3

Easiness Factor Update

new_EF = EF + (0.1 - (3 - quality) × (0.08 + (3 - quality) × 0.02))

# Ensure minimum EF of 1.3
if new_EF < 1.3:
    new_EF = 1.3

Higher quality ratings increase EF → longer intervals
Lower quality ratings decrease EF → shorter intervals

Example Progression

Review # Rating Interval Next Review EF
1 Good 1 day Tomorrow 2.5
2 Good 6 days In 6 days 2.6
3 Good 15 days In 2 weeks 2.7
4 Easy 49 days In 7 weeks 2.8
5 Hard 1 day Tomorrow 2.4

Notice how "Hard" on review #5 resets the interval to 1 day but keeps progress in EF

Why It Works

  • Forgetting Curve - Reviews happen just before you forget
  • Active Recall - Testing memory strengthens it more than re-reading
  • Adaptive - Adjusts to each card's difficulty
  • Efficient - Focus time on cards you're about to forget

Implementation in This App

File: spaced_repetition.py

def calculate_next_review(quality, easiness_factor, interval, repetitions):
    # Update easiness factor
    new_ef = easiness_factor + (0.1 - (3 - quality) * (0.08 + (3 - quality) * 0.02))
    
    # Ensure EF stays within bounds
    if new_ef < 1.3:
        new_ef = 1.3
    
    # Calculate new interval
    if quality < 2:  # Again or Hard
        new_repetitions = 0
        new_interval = 1
    else:  # Good or Easy
        new_repetitions = repetitions + 1
        
        if new_repetitions == 1:
            new_interval = 1
        elif new_repetitions == 2:
            new_interval = 6
        else:
            new_interval = round(interval * new_ef)
        
        if quality == 3:  # Easy bonus
            new_interval = round(new_interval * 1.3)
    
    # Calculate next review date
    next_review = datetime.now() + timedelta(days=new_interval)
    
    return (new_ef, new_interval, new_repetitions, next_review)
